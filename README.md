# 🚦 Bengaluru Parking Intelligence System
**Flipkart GridLock 2.0 — Round 2**

AI-powered parking violation hotspot detection and enforcement prioritization
for the Bengaluru Traffic Police (ASTraM).

---

## Problem Statement
Bengaluru traffic police enforce parking violations reactively through manual patrols,
with no data-driven system to identify where violations are worst, when they peak,
or which zones cause the most congestion. This system uses 298,000+ real violation
records to tell enforcement commanders exactly where to deploy officers, when, and why.

---

## Setup

### 1. Install dependencies
```bash
pip3 install -r requirements.txt
```

### 2. Add your MapMyIndia API key
Edit `.env` file (already created) with your credentials.

### 3. Run the pipeline (one time)
```bash
python3 run_pipeline.py
```

### 4. Launch the dashboard
```bash
streamlit run dashboard/app.py
```

Open http://localhost:8501 in your browser.

## Key Results
- **266 hotspot zones** detected across Bengaluru
- **#1 Critical Zone:** KR Market Junction (82,081 violations, PII: 72.6)
- **Peak violation window:** 10 AM – 4 PM (Afternoons) & 12 AM – 6 AM (Late Night)
- **Worst days:** Sundays (commercial + market spillover)
